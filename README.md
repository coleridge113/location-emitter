# Base Jetpack Compose Project

This is a base project for starting new Jetpack Compose applications. It comes pre-configured with a standard set of libraries and settings to accelerate development.

## Project Configuration

- **Java Version:** 17
- **Kotlin Version:** 2.2.21
- **Android Gradle Plugin:** 8.13.1
- **Compile SDK:** 36
- **Min SDK:** 24
- **Target SDK:** 36

## Included Libraries

This project includes the following libraries by default:

### AndroidX & Jetpack
- Core KTX
- Lifecycle Runtime KTX
- Activity Compose

### Jetpack Compose
- Compose BOM
- Compose UI
- Compose UI Graphics
- Compose UI Tooling Preview
- Compose Material3

### Navigation
- Navigation Compose
- Navigation Fragment KTX
- Navigation UI KTX

### Networking
- Retrofit
- Converter Gson
- Logging Interceptor

### Dependency Injection
- Koin Core
- Koin Android
- Koin AndroidX Compose

### Asynchronous Programming
- Kotlinx Coroutines Core
- Kotlinx Coroutines Android

### Testing
- JUnit
- AndroidX JUnit
- Espresso Core
- Compose UI Test JUnit4
